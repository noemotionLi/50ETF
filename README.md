# 50ETF期权隐含波动率计算以及曲面构建

数据集：2015-2018年50ETF期权日行情

## 功能

数据预处理：筛除数据缺失或者异常的条数

隐含波动率计算：通过B-S model计算期权隐含波动率

统计描述：通过在值情况研究50ETF的隐含波动率分布情况

波动率曲面：构建执行价格、隐含波动率、到期日的3维图形
